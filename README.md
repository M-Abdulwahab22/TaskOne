# Task 1 ::
 
## Create a Java class called Person that includes:

* A field for storing the person's name (String) and age (int).
* A constructor that initializes both fields.
* A method hello() that prints a greeting message (e.g., "Hello, my name is [name] and I am [age] years old").
 
## In your main() method (inside the Person class):
 
* Create two objects of type Person (e.g., person1 and person2).
* Set the values of name and age for each person using the constructor.
* Call the hello() method for each object to print the greeting.
 
## Use comments to describe:
* How the constructor works and how the hello() method is called.
